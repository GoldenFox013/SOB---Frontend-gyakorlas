# Órai munka

### Egy glitch-es felirat kódolása vue keretrendszer használatával

A projekt létrehozása:
```
npm create vite@latest glitch-vue-app -- --template vue
```

A módosított/létrehozott fájlok:
1. src/main.js
2. src/App.vue
3. src/components/GlitchText.vue

Ezután:  
```
npm install
npm run dev
```
